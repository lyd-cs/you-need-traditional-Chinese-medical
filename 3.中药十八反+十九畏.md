# 中药十八反+十九畏



## 十八反：
- 源自：张子和《儒门事亲》
- 本草明言十八反，半蒌贝蔹芨攻乌。藻戟遂芫俱战草，诸参辛芍叛藜芦。
- 意思：
```
乌头（川乌、附子、草乌）反半夏、瓜蒌（全瓜蒌、瓜蒌皮、瓜蒌仁、天花粉）、贝母（川贝、浙贝）、白蔹、白芨；

甘草反甘遂、京大戟、海藻、芫花；



```



## 十九畏
- 源自：刘纯《医经小学》
```
硫黄原是火中精，朴硝一见便相争。
水银莫与砒霜见，狼毒最怕密陀僧。
巴豆性烈最为上，偏与牵牛不顺情。
丁香莫与郁金见，牙硝难合京三棱。
川乌草乌不顺犀，人参最怕五灵脂。
官桂善能调冷气，若逢石脂便相欺。
大凡修合看顺逆，炮爁炙煿莫相依。
```

- 意思：
  - 硫黄畏朴硝，
  - 水银畏砒霜，狼毒畏密陀僧，
  - 巴豆畏牵牛，
  - 丁香畏郁金，
  - 川乌、草乌畏犀角，牙硝畏三棱，
  - 官桂畏石脂，人参畏五灵脂
